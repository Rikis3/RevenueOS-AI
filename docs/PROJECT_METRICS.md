# RevenueOS Project Metrics

This document outlines the scale of the RevenueOS project for portfolio reviewers.

## Data Volume (Synthetic Generation)
*   **Accounts**: 5,000+ B2B companies
*   **Contacts**: 15,000+
*   **Campaigns**: 24 active marketing campaigns across 6 channels
*   **Leads**: 25,000+ generated leads
*   **Opportunities**: 5,000+ created, driving pipeline
*   **Touchpoints**: 75,000+ multi-channel marketing touchpoints
*   **Activities**: 50,000+ logged SDR interactions

## Analytics Architecture
*   **Data Warehouse**: DuckDB Medallion Architecture
*   **Models**: 10 Gold-layer analytical models (Funnel, Attribution, Forecasting, Pipeline, Account Health, etc.)
*   **Metrics Defined**: 45+ unique KPIs modeled in SQL

## AI Intelligence Layer
*   **Copilot Services**: 7 specialized AI engines
*   **LLM Used**: GPT-4o with structured Pydantic outputs
*   **Response Time**: <3 seconds per insight generation

## Tech Stack
*   **Data**: Python, DuckDB, Pandas, Faker
*   **Backend**: FastAPI, Pytest
*   **AI**: LangChain, OpenAI
*   **Frontend**: Next.js 15, Tailwind, ShadCN, Zustand, Recharts, TanStack Query
